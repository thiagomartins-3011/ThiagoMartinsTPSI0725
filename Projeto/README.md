# 🎸 SoundStore — Loja de Instrumentos Musicais

Site de uma loja fictícia de instrumentos musicais, desenvolvido com HTML, CSS e JavaScript.

---

## 📄 Páginas

| Ficheiro | Descrição |
|---|---|
| `index.html` | Página inicial com hero e galeria de imagens |
| `produtos.html` | Catálogo de produtos em grid |
| `contato.html` | Formulário de contato com validação |
| `style.css` | Todos os estilos do site |
| `script.js` | Toda a interatividade em JavaScript |

---

## ⚙️ Funcionalidades JavaScript

- **Menu hamburger** — abre/fecha o menu no mobile com animação
- **Link ativo** — destaca automaticamente a página atual no menu
- **Lightbox** — ao clicar em qualquer imagem da galeria, abre uma janela com detalhes (fecha com o botão ✕, clique fora ou tecla Escape)
- **Validação do formulário** — verifica todos os campos e mostra mensagens de erro antes do envio

---

## 🎨 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis CSS, Flexbox, Grid, media queries para responsividade
- **JavaScript** — sem bibliotecas externas
- **Google Fonts** — Bebas Neue + Outfit

---

## 🚀 Como abrir o site

Não é necessária nenhuma instalação. Basta:

1. Fazer download dos ficheiros
2. Abrir o ficheiro `index.html` diretamente no browser
